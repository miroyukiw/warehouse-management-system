# 仓库管理系统

## 项目简介

本项目为仓库管理系统，主要用于项目部日常仓储业务管理，支持物资入库登记、库存维护、出库领料、批次库存追踪、库存查询编辑以及 Excel 报表生成等功能。

系统采用 Python 开发，基于 Tkinter 构建 Windows 桌面端界面，使用 SQLite 作为本地数据库，结合 pandas、openpyxl、pywin32 实现数据处理与 Excel 文件生成。

## 主要功能

- 入库管理：录入物资入库信息，自动更新库存表和批次库存表。
- 出库管理：从库存表中选择物资出库，自动扣减库存并生成出库记录。
- 库存查询：支持库存表查询、编辑、批次展开和零库存筛选。
- 报表导出：支持生成入库表、出库领料单和项目周报。
- 本地化部署：数据存储于 SQLite 数据库 warehouse.db，不依赖外部服务器。
